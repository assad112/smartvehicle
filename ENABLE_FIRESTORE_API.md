# Enable Cloud Firestore API

## ⚠️ Problem
```
Cloud Firestore API has not been used in project cariot-f6f84 before or it is disabled.
```

## ✅ Quick Solution

### Method 1: Direct Link
1. Click on the following link:
   ```
   https://console.developers.google.com/apis/api/firestore.googleapis.com/overview?project=cariot-f6f84
   ```
2. Click **Enable**
3. Wait a few minutes for activation

### Method 2: From Firebase Console
1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Select your project: **cariot-f6f84**
3. Go to **Firestore Database**
4. If it doesn't exist, click **Create database**
5. Select **Start in test mode**
6. Choose a location (e.g., `us-central1`)
7. Click **Enable**

### Method 3: From Google Cloud Console
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Select the project: **cariot-f6f84**
3. Go to **APIs & Services** > **Library**
4. Search for **Cloud Firestore API**
5. Click **Enable**

## ⏱️ After Activation

- Wait 2-5 minutes for the API to be activated
- Restart the application
- Firestore should now work

## ✅ Verification

After activation, the error message should disappear and the database should work correctly.
